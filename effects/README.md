# Effects Examples

Public reference implementations live here.

## Available examples

- `volumetric-lighting-webgpu`: volumetric lighting scene with a WebGPU-first path and a documented fallback contract

## Preview

From the repository root, run:

```bash
python3 -m http.server 4173 -d .
```

Then open:

```text
http://localhost:4173/effects/volumetric-lighting-webgpu/
```

Do not treat `file://` as a supported preview mode for these zero-build demos.
